Entendendo Cliente X Server

  • Client: Client(Cliente)É aqueles que consome os conteúdos disponíveis na internet com algum tipo de dispositivo como computadores e celulares entre outros, funciona da seguinte forma, os computadores são interligados por uma rede chamada internet que serve para se comunicar entre cliente e servidor, essa interação acontece pelos navegadores(Browsers) que, aliás tem vários tipos, mas seguindo uma linguagem de programação padrão W3C,porem, sempre tem uma diferença ou outra de interpretação.

    Bem, essa comunicação funciona da seguinte maneira: o usuário entra no navegador de sua preferência, solicita o serviço, qual serviço é esse? , esse serviço se da quando você digita o endereço de um site, ele manda uma solicitação através da internet até o servidor a qual aquele link pertence e o servidor retorna arquivos, aplicações web, mídias, etc, pronto para você interagi(consumir o conteúdo) com aquele site web.
    Aplicações Web: Aplicações Web: É um software executado em um navegador da Web. As empresas precisam trocar informações e prestar serviços remotamente. Elas usam aplicações Web para se conectar com os clientes de forma conveniente e segura.
    Sendo assim, essas aplicações precisam ser compatíveis com diferentes tipos de dispositivos, que hoje em dia são vários. Por exemplo, notebook,dispositivos móveis, televisões, computadores, etc.

    Dispositivos Móveis: São eles os celulares, tablet, relógios smartwatch, bastante utilizados no dia dia da população.pesquisas mostra que, de 10 pessoas, 9 delas possuem ou utilizam algum tipo desses dispositivos e a tendência é só aumentar. Aparelhos móveis esses que têm as mesmas funções de um computador e bem mais práticos e acessíveis.

  • Server: Um servidor é uma máquina que serve de apoio para outros dispositivos, que estejam conectados à internet ou não. Um servidor local é quando você tem vários computadores se conectando a um computador central mesmo sem internet, tipo uma empresa onde tem um computador central que armazena documentos, pastas, imagens, etc. e pode ser acessível por vários funcionários de acordo com suas credenciais.
    Com conexão à internet, o servidor tem as mesmas funções de servir o usuário que faz a requisição por meio de aplicativos ou aplicações web,independente de onde se localiza o usuário.
    Servidores tendem a ser mais rápidos, com maior capacidade de armazenamento,os preços são mais elevados.
    Há vários tipos de servidor sendo:
    • Arquivos
    • Streaming
    • Web
    • Segurança
    • E-mail

    Há vários tipos de servidores e, com tantas variações de serviços,exigem certos componentes, tanto hardwares como softwares. Dependendo de qual serviço você for utilizar e com qual frequência vai ser acessado, tem que colocar hardwares de acordo e softwares também, e até o sistema operacional de acordo. Sim, o SO de um servidor é diferente de um computador normal.
    • Tipos de servidores:
      • Proxy: Serve para restringir/autenticar acesso de usuário a certo destino.
      • Firewall: É uma parede de fogo contra programas maliciosos ou usuário externo.
      • WebServer: Utilizado para sites.
      • E-mailServer: Utilizado para E-mail.
      • DataBase Server: Banco de dados de um servidor.
      • DNS: Domain Name Service: É um servidor que gera um ip para cada domínio criado em rede, tipo um RG.

    • Hospedagem de domínio(site): É quando você cria um site e quer colocar na web de maneira que qualquer usuário possa acessar remotamente de onde estiver por meio de um navegador que faz uma solicitação em um Web server.

Html na Prática

HTML é a linguagem de marcação padrão para a criação de páginas da Web.

    O que é HTML?
    • HTML significa linguagem de marcação de hipertexto
    • HTML é a linguagem de marcação padrão para criação de páginas da Web
    • HTML descreve a estrutura de uma página da Web
    • HTML consiste em uma série de elementos
    • Os elementos HTML informam ao navegador como exibir o conteúdo
    • Os elementos HTML rotulam partes do conteúdo como “este é um título”, “este é um parágrafo”, “este é um link”, etc.

Entregando seu projeto

Bem, após ter feito o projeto, está na hora de apresentá-lo e uma boa plataforma para publicá-lo é o GitHub.
Como faremos para expor o seu projeto? Primeiramente, precisamos entender como funciona o GitHub e o Git.

  • GitHub: É uma nuvem onde você compartilha e armazena seus trabalhos para acessar de qualquer lugar e, no caso, trabalhe com uma equipe. É possível que a pessoa tenha acesso ao projeto, permitindo que ela acesse, analise e modifique seu conteúdo.

  • Git: É uma ferramenta que serve para salvar o seu código, pois futuramente você vai fazer alterações, trabalhar com pessoas, adicionar ou retirar elementos e algumas vezes pode dar errado. Sendo assim, ele cria como se fosse um ponto seguro daquele documento que você está adicionando ou modificando, para que, caso algo de errado aconteça, possa voltar para onde estava funcionando, tudo ok.
